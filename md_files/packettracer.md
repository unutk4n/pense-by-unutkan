Packet Tracer has the ability to create four different types of files. These file types are used for different purposes and include: .pka, .pkt, .pksz, and .pkz.


The PKA one is the most popular one. This file type usually has a instruction window. It's mainly for learners because it has some questions and answers. 

The PKT one has a simulated network inside. It can also have graphic background images inside. 


The PKSZ one is the comes with Tutored Activity. More friendly than the PKA.

The PKZ one is old and is not being used anymore. It was used to embed photos insisde but we can now do it with the regular one.


Packet Tracer Media Object >>  PTMOs can be either a .pkt or .pka file. A PTMO will show up inline with a question on your quiz, module exam, or some other type of graded assessment.


Packet Tracer Skills Assessment >> You can encounter PTSAs at a mid-point in a course or at the end of the course. PTSAs are summative skill assessments in which you demonstrate your skill by implementing the technologies that you have learned. PTSAs are standalone assessments, have their own grading engine.
